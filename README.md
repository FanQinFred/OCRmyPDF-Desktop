# OCRmyPDF桌面端
鉴于很多使用者不会配置环境，我们在OCRmyPDF的基础上，集成了所需环境，并使用Electron开发了桌面端。

<img decoding="async" src="https://caj2pdf-cdn.cdn.bcebos.com/pdfocr/oc1nf-cxfc2.gif" width="100%">

***

## 安装



### [桌面端安装包点击下载](https://fanfan-pdf-ocr.caj2pdf.cn/)


虽然是使用Electron多平台支持编写的此项目，但是目前仅支持Windows平台。同时由于集成OCR多个环境和32位操作系统依赖，安装包目前较大。

上述安装包链接通过百度云进行的托管，并且开启了CDN，还请不要重复下载，流量费用较高。

***
